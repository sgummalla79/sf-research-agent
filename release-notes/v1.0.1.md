# v1.0.1 — 2026-05-14

## Features

- Skill framework — declarative pipeline engine replacing graph/builder
- Auth0 multi-tenant authentication with httpOnly cookie sessions
- Per-user DB isolation — user_id on all tables
- Per-user HKDF encryption for API keys at rest
- Skill install/uninstall, skill-level versioning
- Post-completion chat, session fork, sidebar context menus
- Comprehensive /health endpoint with per-check diagnostics
- Auto-increment versioning — VERSION file + bump type selector
